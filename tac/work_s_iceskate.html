Novena Ice Skate

Novena Ice Skate is a part of the Christmas campaign for Novena. It was run as a badge hunt contest linked to facebook with facebook Connect.

The idea was tied up with Novena installing an ice skating rink for the holiday season. Visitors will need to browse through Novena's website to hunt for the badge, and once found, to install the facebook application. At the end of the campaign, users with the most badge win.

Project Completed: November 2009

Industry: Retail
