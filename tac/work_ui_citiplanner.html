CitiPlanner

Citibank ITO team has created a profiling application to allow Relationship Manager (RM) to understand their customer portfolio and at the same time upsell them with more relevant products. This tool was used as a client interface between both RM and the clients.

Citibank approached us to improve the user experience to encourage more usage by the RMs, on top of having a more professional look. Moving forward, Citibank is planning to allow their clients to use this tool for their personal profile assessment.

We approached this project by using several UX methods such as; paper-prototyping and UX wireframing. The production was executed in Flash to ensure smooth animation and transition.

Project Completed: September 2009

Industry: Banking and Finance
